# Changelog

## v2.0 - 2026-02-04 (正式发布)

### 新增
- ✨ 完全基于libvgm的ADPCM渲染
- ✨ 批量转换脚本支持
- ✨ 示例文件（输入/输出）
- ✨ 完整的发布文档

### 改进
- 🎵 优化硬件兼容性（22.05kHz采样率）
- 🎵 自动采样率转换（22.05kHz → 44.1kHz）
- 🎵 音量平衡优化（FM 1/6, PCM 150%）
- 📦 减少50% PCM写入命令

### 修复
- 🐛 修复播放速度问题（采样率转换）
- 🐛 修复PCM无声音问题（DAC立体声设置）
- 🐛 删除不准确的手动ADPCM解码器

### 移除
- ❌ 删除ADPCMDecoder.cpp/h（改用libvgm）
- ❌ 删除adpcm2wav工具（改用vgm2wav_adpcm_only）
